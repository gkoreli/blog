# Hello World

This is the first post on my blog. I'm Goga Koreli, and I build tools for developers.

I created [@nisli/core](https://www.npmjs.com/package/@nisli/core), a zero-dependency reactive web component framework, and [backlog-mcp](https://github.com/gkoreli/backlog-mcp), an MCP server that lets AI agents manage their own task backlogs.

This blog is built with `@nisli/core` — dogfooding my own framework. I'll be writing about:

- Building open source tools
- The design decisions behind my projects
- What I learn along the way

Let's go.
