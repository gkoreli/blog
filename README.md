# gkoreli.com

Personal blog by Goga Koreli — built with [@nisli/core](https://www.npmjs.com/package/@nisli/core).

Writing about building tools, open source, and the craft of software.

## Development

```bash
npm install
npm run dev
```

## Stack

- **Framework**: [@nisli/core](https://github.com/gkoreli/backlog-mcp/tree/main/packages/framework) — zero-dependency reactive web components
- **Content**: Markdown posts in `posts/`
- **Hosting**: GitHub Pages + Cloudflare
